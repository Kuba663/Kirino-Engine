# TypeUtils

> Relevant classes:
> <br>· `com.cleanroommc.kirino.utils.TypeUtils`

It provides a set of utility functions for primitive type and wrapped primitive type related operations.

Common usage pattern:
- `TypeUtils.looseTypeCheck(...) // ignores the difference between primitives and wrapped primitives`
- `TypeUtils.toWrappedPrimitive(...)`
- `TypeUtils.toPrimitive(...)`
