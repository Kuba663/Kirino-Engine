# MinecraftResourceUtils

> Relevant classes:
> <br>· `com.cleanroommc.kirino.utils.MinecraftResourceUtils`

It provides you the easiest way to load resources via `ResourceLocation`.

Common usage pattern:
- `MinecraftResourceUtils.readText(...)`
- `MinecraftResourceUtils.getInputStream(...)`

> Note:
> This resource loader works for `cleanroomClient` and unit tests too
> under Cleanroom dev env (Cleanroom itself not the mod template)
